錯誤代碼解釋

錯誤代碼 400 - Bad Request

由於明顯的客戶端錯誤(例如，格式錯誤的請求語法，太大的大小，無效的請求訊息或欺騙性路由請求)，伺服器不能或不會處理該請求。

錯誤代碼 403 - Forbidden

伺服器已經理解請求，但是拒絕執行它。有可能您被該伺服器封鎖拒絕存取了。

您可以: 嘗試聯絡伺服器管理員。

錯誤代碼 404 - Page Not Found

請求失敗，請求所希望得到的資源未被在伺服器上發現，但允許使用者的後續請求。

您可以: 嘗試輸入正確的網址。

錯誤代碼 429 - Too Many Requests

您在給定的時間內傳送了太多的請求。請稍等片刻後再傳送請求。

錯誤代碼 500 - Internal Server Error

伺服器遇到了一個未曾預料的狀況，導致了它無法完成對請求的處理。

您可以: 稍後片刻再試一次，或嘗試聯絡伺服器管理員。
